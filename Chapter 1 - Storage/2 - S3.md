# Object Storage (S3-Compatible) – Client Perspective

Object storage systems expose a simple HTTP-based API for storing and retrieving data as objects.

From a client perspective, S3-compatible storage provides a unified way to interact with data regardless of the underlying implementation or vendor.

:warning: **Note:**  
This chapter is written strictly from the **client point of view**.  
We focus on how applications interact with the system not how it is implemented internally.  
We refer to **S3 and S3 compatible APIs**, not necessarily a specific provider like AWS.
